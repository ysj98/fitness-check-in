# 运动打卡小程序

## 核心功能

- 微信登录
- 每日打卡
- 每日多次打卡
- 今日记录
- 最近打卡记录
- 月份切换
- 月度打卡日历
- 连续打卡天数
- 打卡动态效果
- 微信头像获取
- 微信昵称填写
- 个人资料维护

## 打卡页

- 当前日期
- 连续天数
- 立即打卡
- 打卡成功动画
- 上月切换
- 下月切换
- 月度概览
- 今日记录删除
- 最近 12 次记录

## 我的页

- 头像地址
- 昵称
- 微信头像选择
- 微信昵称输入
- 性别
- 生日
- 资料保存

## 后端功能

- 微信登录换码
- JWT 鉴权
- 用户资料查询
- 用户资料更新
- 今日打卡统计
- 月度打卡统计
- 连续打卡统计
- 最近打卡查询
- 打卡记录创建
- 打卡记录删除

## 部署能力

- Node.js API
- Fastify 服务
- Prisma 数据模型
- MySQL 存储
- Dockerfile
- Docker Compose
- Nginx HTTPS 反向代理
- 微信小程序生产构建
